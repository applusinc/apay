`Android`

`iOS`

`Web`

  
## Projeyi Bilgisarınıza İndirin

Projeyi klonlayın

```bash
  git clone https://link-to-project
```

Proje dizinine gidin

```bash
  cd apay
```

Flutter için gerekli paketleri denetleyin

```bash
  flutter doctor -v
```

Proje kaynaklarını indirin

```bash
  flutter pub get
```

Projeyi Android için build edin

```bash
  flutter build apk
```
